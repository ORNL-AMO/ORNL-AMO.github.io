14 To edit the CMakeCache: `ccmake .` from project root directory, enable desired build options, then `make` to build.
15 
16 To build C++ unit tests directly: `cmake --build . --target amo_tools_suite_tests` from project root directory.
17 
18 To create an installer package: `ccmake .` from project root directory, then enable the `BUILD_PACKAGE` flag, then `cmake ./` then `make package`
